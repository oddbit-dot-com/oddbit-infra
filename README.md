# Oddbit Infrastructure

## Frontend services

### Keycloak

[Keycloak] is an identity provider.

[keycloak]: https://www.keycloak.org/

### OpenBao (Vault)

[OpenBao] is a secret management service. It is a fork of [Vault].

[openbao]: https://openbao.org/
[vault]: https://www.hashicorp.com/en/products/vault

## Backend services

### Postgres

Used as the storage backend for Keycloak.

### Maddy

[Maddy] is a mail server used to provide outbound SMTP for Postgres.

[maddy]: https://maddy.email/
